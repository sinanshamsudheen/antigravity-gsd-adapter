# Changelog

All notable changes to this adapter will be documented in this file.

## [1.0.0] - 2026-01-16

### Added
- Initial release of Antigravity GSD Adapter
- Automated setup script (`setup-gsd.sh`) for one-command installation
- Comprehensive usage guide (GUIDE.md) covering all 20 workflows
- Sample PRD and example project structure
- Path migration from `.claude` to `.agent`
- Project-level workflow symlinks
- Installation to `~/.agent/get-shit-done/`

### Changed
- Adapted all 20 workflow files for Antigravity compatibility
- Updated 21 template files with correct path references
- Modified workflow references from ClaudeCode to Antigravity

### Documentation
- Created README.md with quick start guide
- Added GUIDE.md with detailed workflow instructions
- Included example PRD (task management SaaS)
- Added CONTRIBUTING.md for community guidelines
- Created LICENSE (MIT) with proper attribution
